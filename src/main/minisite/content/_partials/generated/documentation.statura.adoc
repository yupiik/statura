== Configuration

* `-.checks.$index.connection.host` (`__CHECKS_INDEX_CONNECTION_HOST`): Host to connect to.
* `-.checks.$index.connection.port` (`__CHECKS_INDEX_CONNECTION_PORT`) (default: `0`): Port to connect to.
* `-.checks.$index.connection.proxy.host` (`__CHECKS_INDEX_CONNECTION_PROXY_HOST`): Proxy host.
* `-.checks.$index.connection.proxy.port` (`__CHECKS_INDEX_CONNECTION_PROXY_PORT`) (default: `0`): Proxy port.
* `-.checks.$index.connection.proxy.type` (`__CHECKS_INDEX_CONNECTION_PROXY_TYPE`) (default: `"DIRECT"`): Proxy type: HTTP, SOCKS (SOCKS5), SOCKS4 (or SOCKS4a), or DIRECT.
* `-.checks.$index.connection.timeout` (`__CHECKS_INDEX_CONNECTION_TIMEOUT`) (default: `"PT10S"`): Custom (java duration formatted) timeout for this check.
* `-.checks.$index.dns.expectedAddresses` (`__CHECKS_INDEX_DNS_EXPECTEDADDRESSES`): Optional list of expected resolved IP addresses.
* `-.checks.$index.dns.hostname` (`__CHECKS_INDEX_DNS_HOSTNAME`): Hostname to resolve.
* `-.checks.$index.dns.timeout` (`__CHECKS_INDEX_DNS_TIMEOUT`) (default: `"PT10S"`): Custom (java duration formatted) timeout for this check.
* `-.checks.$index.http.assertions.$index.expected-value` (`__CHECKS_INDEX_HTTP_ASSERTIONS_INDEX_EXPECTED_VALUE`) (default: `"true"`): Expected value.
* `-.checks.$index.http.assertions.$index.json-pointer` (`__CHECKS_INDEX_HTTP_ASSERTIONS_INDEX_JSON_POINTER`) (default: `"/"`): JSON-Pointer to extract.
* `-.checks.$index.http.assertions.$index.operator` (`__CHECKS_INDEX_HTTP_ASSERTIONS_INDEX_OPERATOR`) (default: `io.yupiik.statura.check.HttpCheckConfiguration.AssertionOperator.EXISTS`): Expected value.
* `-.checks.$index.http.body` (`__CHECKS_INDEX_HTTP_BODY`) (default: `""`): For _write_ methods, the payload to send.
* `-.checks.$index.http.expected-status` (`__CHECKS_INDEX_HTTP_EXPECTED_STATUS`) (default: `200`): Expected response status.
* `-.checks.$index.http.followRedirects` (`__CHECKS_INDEX_HTTP_FOLLOWREDIRECTS`) (default: `false`): Should redirects be followed.
* `-.checks.$index.http.headers` (`__CHECKS_INDEX_HTTP_HEADERS`) (default: `java.util.Map.of()`): HTTP headers to set.
* `-.checks.$index.http.method` (`__CHECKS_INDEX_HTTP_METHOD`) (default: `"GET"`): HTTP method to use.
* `-.checks.$index.http.ssl-certificates` (`__CHECKS_INDEX_HTTP_SSL_CERTIFICATES`): Trusted CA certificates in PEM format for custom SSL trust.
* `-.checks.$index.http.ssl-client-certificate` (`__CHECKS_INDEX_HTTP_SSL_CLIENT_CERTIFICATE`): Client certificate in PEM format for mTLS.
* `-.checks.$index.http.ssl-client-private-key` (`__CHECKS_INDEX_HTTP_SSL_CLIENT_PRIVATE_KEY`): Client private key in PEM format (PKCS#8) for mTLS.
* `-.checks.$index.http.timeout` (`__CHECKS_INDEX_HTTP_TIMEOUT`) (default: `"PT30S"`): Custom (java duration formatted) timeout for this check.
* `-.checks.$index.http.url` (`__CHECKS_INDEX_HTTP_URL`): URL to call.
* `-.checks.$index.http.version` (`__CHECKS_INDEX_HTTP_VERSION`) (default: `java.net.http.HttpClient.Version.HTTP_1_1`): HTTP protocol version.
* `-.checks.$index.jdbc.driver` (`__CHECKS_INDEX_JDBC_DRIVER`): JDBC driver (if needed, recent drivers can be loaded from the classpath).
* `-.checks.$index.jdbc.password` (`__CHECKS_INDEX_JDBC_PASSWORD`): JDBC password, prefer using secret injection for this.
* `-.checks.$index.jdbc.timeout` (`__CHECKS_INDEX_JDBC_TIMEOUT`) (default: `"PT30S"`): Custom (java duration formatted) timeout for this check.
* `-.checks.$index.jdbc.url` (`__CHECKS_INDEX_JDBC_URL`): JDBC URL.
* `-.checks.$index.jdbc.username` (`__CHECKS_INDEX_JDBC_USERNAME`): JDBC username.
* `-.checks.$index.jdbc.validation-query` (`__CHECKS_INDEX_JDBC_VALIDATION_QUERY`) (default: `"SELECT 1"`): SQL query to validate the connection.
* `-.checks.$index.name` (`__CHECKS_INDEX_NAME`): Check name (or an internal one is generated).
* `-.checks.$index.type` (`__CHECKS_INDEX_TYPE`) (default: `io.yupiik.statura.CheckExecutor.CheckType.HTTP`): Type of the validation.
* `-.checks.$index.x509.allow-self-signed` (`__CHECKS_INDEX_X509_ALLOW_SELF_SIGNED`) (default: `false`): Whether to accept self-signed certificates.
* `-.checks.$index.x509.expected-fingerprint` (`__CHECKS_INDEX_X509_EXPECTED_FINGERPRINT`): Expected SHA-256 fingerprint (hex, colon-separated or continuous).
* `-.checks.$index.x509.expected-issuer` (`__CHECKS_INDEX_X509_EXPECTED_ISSUER`): Expected Issuer DN regex pattern.
* `-.checks.$index.x509.expected-sans` (`__CHECKS_INDEX_X509_EXPECTED_SANS`): Expected Subject Alternative Names.
* `-.checks.$index.x509.expected-subject` (`__CHECKS_INDEX_X509_EXPECTED_SUBJECT`): Expected Subject DN regex pattern.
* `-.checks.$index.x509.host` (`__CHECKS_INDEX_X509_HOST`): Hostname to connect to.
* `-.checks.$index.x509.min-days-until-expiration` (`__CHECKS_INDEX_X509_MIN_DAYS_UNTIL_EXPIRATION`) (default: `0`): Minimum days before certificate expiration.
* `-.checks.$index.x509.port` (`__CHECKS_INDEX_X509_PORT`) (default: `443`): Port to connect to.
* `-.checks.$index.x509.sni` (`__CHECKS_INDEX_X509_SNI`): SNI hostname (defaults to `host`).
* `-.checks.$index.x509.timeout` (`__CHECKS_INDEX_X509_TIMEOUT`) (default: `"PT30S"`): Custom (java duration formatted) timeout for this check.
* `-.checks.$index.x509.verify-hostname` (`__CHECKS_INDEX_X509_VERIFY_HOSTNAME`) (default: `true`): Verify hostname matches certificate SAN/CN.
* `-.default-timeout` (`__DEFAULT_TIMEOUT`) (default: `"PT30S"`): Default timeout per check.
* `-.execution-timeout` (`__EXECUTION_TIMEOUT`) (default: `"PT15M"`): Default timeout for the whole execution.
* `-.max-concurrent-per-type` (`__MAX_CONCURRENT_PER_TYPE`) (default: `java.util.Map.of()`): Max parallel checks per type.
* `-.opentelemetry.endpoint` (`__OPENTELEMETRY_ENDPOINT`): OpenTelemetry collector HTTP endpoint for metrics.
* `-.opentelemetry.headers` (`__OPENTELEMETRY_HEADERS`) (default: `java.util.Map.of()`): Headers to send to respect OpenTelemetry collector endpoint.
* `-.opentelemetry.httpVersion` (`__OPENTELEMETRY_HTTPVERSION`) (default: `java.net.http.HttpClient.Version.HTTP_1_1`): HTTP protocol version for the collector connection.
* `-.opentelemetry.protocol` (`__OPENTELEMETRY_PROTOCOL`) (default: `io.yupiik.statura.otel.OpenTelemetry.Protocol.JSON`): Serialization protocol: JSON or PROTOBUF.
* `-.opentelemetry.resource-attributes` (`__OPENTELEMETRY_RESOURCE_ATTRIBUTES`) (default: `java.util.Map.of("service.name", "statura", "service.version", "1.0")`): Resource attributes (e.g. service.name, service.version).
* `-.opentelemetry.scope-attributes` (`__OPENTELEMETRY_SCOPE_ATTRIBUTES`) (default: `java.util.Map.of("name", "statura", "version", "1.0")`): Instrumentation scope attributes (e.g. name, version).
* `-.opentelemetry.ssl-certificates` (`__OPENTELEMETRY_SSL_CERTIFICATES`): Trusted CA certificates in PEM format for custom SSL trust.
* `-.opentelemetry.ssl-client-certificate` (`__OPENTELEMETRY_SSL_CLIENT_CERTIFICATE`): Client certificate in PEM format for mTLS.
* `-.opentelemetry.ssl-client-private-key` (`__OPENTELEMETRY_SSL_CLIENT_PRIVATE_KEY`): Client private key in PEM format (PKCS#8) for mTLS.
* `-.thread-per-type` (`__THREAD_PER_TYPE`) (default: `java.util.Map.of(io.yupiik.statura.CheckExecutor.CheckType.JDBC.name(), "8")`): Thread mode: <0 synchronous, 0 virtual threads, >0 fixed pool size.
